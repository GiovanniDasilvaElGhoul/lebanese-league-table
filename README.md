# Lebanese League Table

## Overview

Lebanese League Table is a Python-based project developed during my M1 studies.  
The goal of this project is to manage, analyze, and visualize the Lebanese football league table in a clear and interactive way.

The project helps users explore team standings, match results, points, wins, draws, losses, and other league-related statistics.

## Project Objective

The main objective of this project is to transform raw football league data into a structured and meaningful dashboard.

Instead of only displaying static tables, the project focuses on making the data easier to understand through visualizations, statistics, and interactive features.

## Features

- Display the Lebanese league table
- Show team rankings and standings
- Analyze wins, draws, losses, and points
- Visualize football statistics using charts
- Provide an interactive dashboard interface
- Help users better understand team performance during the season

## Technologies Used

- Python
- Streamlit
- Pandas
- Matplotlib / Plotly
- CSV dataset
